# Story: Strip all Context Bonsai footprint from the pi-mono fork

**Epic:** Relocate Pi Context Bonsai Out Of The pi-mono Fork
**Size:** Small
**Dependencies:** Stories 1, 2, and 3 (all verified green — this story deletes the fork's only copy of the code, the integration tests, and the harness).

## Story Description

Remove every Context Bonsai artifact from the `Vibecodelicious/pi-mono` fork so the fork carries no bonsai code. Research established the fork's bonsai branch is purely additive, so this is a clean removal: delete the additions and the fork returns to upstream-equivalent state for bonsai purposes.

This story is gated: it MUST NOT begin until Stories 1–3 are verified green, because it deletes the in-tree package, the integration tests, and the SDK harness Story 1 added — the originals of everything relocated. If the relocated copies are not proven first, this story destroys working coverage.

The footprint to remove:
- `pi/packages/context-bonsai/` — the in-tree extension package.
- `pi/.pi/extensions/context-bonsai` — the discovery symlink (committed via `4d4d092d`).
- The Context Bonsai test-only alias in `pi/packages/coding-agent/vitest.config.ts`.
- `pi/packages/coding-agent/test/suite/context-bonsai/` — the integration tests and the SDK harness Story 1 added.
- The `!.agents/plans/` and `!.agents/plans/**` un-ignore lines in `pi/.gitignore`.
- `pi/.agents/plans/epic-port-context-bonsai/` and `pi/.agents/research/pi-e2e-interaction-baseline.md` — the fork's bonsai plan/research docs (canonical copies are retained in the parent repo at `.agents/plans/epic-pi-port/`).
- The `@mariozechner/pi-context-bonsai` entries in `pi/package-lock.json` (regenerated by `npm install`).

The shared `pi/packages/coding-agent/test/suite/harness.ts` and `pi/packages/coding-agent/test/utilities.ts` are NOT bonsai footprint and are left untouched — Story 1 did not modify them.

The `pi` submodule is retained in the parent repo (user decision) as a pinned pi-mono reference; it simply carries no bonsai code afterward.
